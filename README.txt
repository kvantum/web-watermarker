Run in terminal:
    python3 server.py

Open a browser and go to the address:
    localhost:8000/index.html

- Browse an image file.
- Enter width and height of the image box to resize your image. The image will not be resized if its size is less than an image box. In this case the image will be pasted into the center of the image box.
- Browse a watermark image file (optional).
- Type a text to add as a watermark over the image box.
- Enter an angle of the watermark text inclination.
- Enter watermark opacity (floating point number from 0 to 1). This value will be applied to the text and image watermarks. The default value is 0.25.
